017. v-show

v-show 是簡單的切換元素的 CSS 屬性 display。元素會始終渲染並保持在 DOM 中。

注意 v-show 不支援 template 語法。

Hello!

一般來說， v-if 有更高的切換消耗而 v-show 更高的初始渲染消耗。因此，如果需要頻繁切換使用 v-show 較好，如果在運行時條件不大可能改變則使用 v-if 較好。
